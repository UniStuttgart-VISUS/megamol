# btf-peek README

Allows to jump quickly between btfs and the actual shader code.

## Features

Should make [F12] / [ALT]-[F12] or 'Peek Definition' available on paths inside a btf file.

## How to install

You need node.js installed.

Add the vsce package:

    npm install -g vsce

generate the vsix:

    vsce package

install from that.
